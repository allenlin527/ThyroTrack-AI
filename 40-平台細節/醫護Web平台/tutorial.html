常見問題

如何重置我的平台密碼？

+

您可以通過以下步驟重置密碼：

  1. 在登入頁面點擊「忘記密碼」
  2. 輸入您的註冊電子郵件地址
  3. 您將收到一封包含重置鏈接的電子郵件
  4. 點擊鏈接並設置新密碼

若未收到電子郵件，請檢查垃圾郵件資料夾或聯絡系統管理員。

如何導出患者數據進行研究分析？

+

要導出患者數據，您需要：

  1. 進入「數據分析與報表中心」
  2. 點擊「研究數據導出」區塊
  3. 選擇需要的數據類型與時間範圍
  4. 填寫導出目的說明
  5. 選擇輸出格式（CSV、Excel、SPSS等）
  6. 點擊「導出數據」

注意：所有導出的數據都會自動去識別化，以保護患者隱私。

如何處理患者上傳的異常數據與警示？

+

處理警示的標準流程如下：

  1. 在儀表板或警示管理頁面查看待處理警示
  2. 點擊警示卡片或列表項進入詳情
  3. 檢查患者數據與症狀描述
  4. 使用AI輔助決策系統獲取建議（可選）
  5. 選擇適當的處理方式：
    • 標記為已閱覽
    • 安排遠距諮詢
    • 記錄臨床決策與處理意見
    • 提高或降低患者風險等級

平台支援哪些瀏覽器？

+

本平台支援以下瀏覽器的最新版本：

  • Google Chrome
  • Microsoft Edge
  • Mozilla Firefox
  • Apple Safari

為獲得最佳使用體驗，建議使用 Google Chrome 或 Microsoft Edge 的最新版本。

我的問題沒有在此列出，如何獲取支援？

+

如果您的問題沒有在常見問題中列出，您可以：

  • 點擊右下角的「支援」圖標，與技術支援團隊聊天
  • 發送電子郵件至 support@thyroid-platform.tw
  • 在系統設定頁面提交意見回饋
  • 撥打技術支援熱線：(02) 2345-6789，服務時間為週一至週五 9:00-18:00
